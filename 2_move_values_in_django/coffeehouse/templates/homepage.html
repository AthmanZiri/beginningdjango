Django HTTP handling

  • 302 - Permanent redirect /stores/1/?latitude=90&longitude=90 (Latitude 90 and longitude 90, redirects to maps.google.com)
  • 400 - Bad request /stores/1/?latitude=ABC&longitude=DEF (Latitude and/or Longitude not numbers)
  • 404 - Not found /stores/1/?hours=lundi (Unknown day to get hours)
  • 500 - Internal server error /stores/1/?latitude=92.71&longitude=-117.16 (Latitude out of range -90 to 90)
  • 500 - Internal server error /stores/1/?latitude=32.71&longitude=-187.16 (Longitude out of range -180 to 180)
